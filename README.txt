WordPress Mobile Edition README
version 1.8, 2006-03-02

This hack will provide mobile device friendly interface for your blog.

Step 1.

Put the wp-mobile.php and wp-mobile.css files in your wordpress folder.


Step 2.

Put the wp-mobile.plugin.php file in your wp-content/plugins/ folder.


Step 3.

Activate the plugin on your plugins page in the WordPress Admin screens.


That's it!
